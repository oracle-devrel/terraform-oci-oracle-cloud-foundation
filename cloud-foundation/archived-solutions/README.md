## About This Folder

The `archived-solutions` folder contains older Oracle Cloud Foundation Terraform solutions that remain publicly available for reference, but are no longer part of the active solution portfolio.

These solutions have been moved here because they are not currently planned for active maintenance, modernization, or regular updates. Some of them may still be useful as examples, historical references, or starting points for specific use cases, but they should not be treated as the primary recommended implementation for new deployments.

This folder helps keep the main `solutions` folder focused on active, relevant, and maintained solutions, while still preserving access to previous work.

A solution may be archived because:

- it is based on an older architecture pattern;
- it has been replaced by a newer or more relevant solution;
- it requires updates before being promoted again;
- it has unclear or inactive ownership;
- it overlaps with another active solution;
- it is kept mainly for reference or historical continuity.

Archived does not mean deleted or private. The content remains available in the repository, but it should be reviewed carefully before reuse.

Before using or promoting any archived solution, the solution should be validated against current OCI services, Terraform versions, OCI provider versions, security requirements, and documentation standards.
